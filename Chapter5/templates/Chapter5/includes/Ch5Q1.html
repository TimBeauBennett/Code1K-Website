Chapter 5 - Numeric Types

1: What is the value of the expression 2 * (3 + 4) in Python?

- My Answer: 14. Because expressions in parentheses are evaluated first.

- Mark's Answer: The value will be 14, the result of 2 * 7, because the parentheses force the addition to happen before the multiplication.

2: What is the value of the expression 2 * 3 + 4 in Python?

- My Answer: 10. Because multiplication is higher than addition in order of operations.

- Mark's Answer: The value will be 10, the result of 6 + 4. Python's operator precedence rules are applied in the absence of parentheses, and multiplication has higher precedence than (ie it happens before) addition.

3: What is the value of the expression 2 + 3 * 4 in Python?

- My Answer: 14. Same reason as for Q2.

- Mark's Answer: This expression yields 14, the result of 2 + 12, for the same precendence reasons as in the prior question.

4: What tools can you use to find a number's square root, as well as its square?

- My Answer: To square a number, you canhard code the multiplication (eg 3 * 3) or you can use the exponent (eg 3 ** 2). To get the square root, you can import math and use the sqrt() function. alternatively, you can raise the variable to a half (eg 3 ** .5) which is mathematically equivalent. There was another way Mark mentioned which I forget, but I remember that it was basically a duplication of the latter of those I mentioned.

- Mark's Answer: Functions for obtaining the square root, as well as pi, tangents, and more, are available in the imported math module. To find a number's square root, import math and call math.sqrt(N). To get a number's square, use either the exponent expression X ** 2 or the build-in function pow(X, 2). Either of these last two can also compute the square root when given a power of 0.5 (eg, X ** .5).

5: What is the type of the result of the expression 1 + 2.0 + 3?

- My Answer: The answer would be 6.0, and the type a floating-point number. Python will automatically convert types up to the most complex before running the math, and output that more complex type, in this case, a floating point.

- Mark's Answer: The result will be a floating point number: the integers are converted up to floatin point, the most complex type in the expression, and floating point math is use to evaluate it.

6: How can you truncate and round a floating-point number?

- My Answer: By importing the math module, trunc(x) will truncate the number, and round(x) will round it (the latter is built in). Truncate just drops the numbers after the decimal, while round aderes to the mathematical operation

- Mark's Answer: The int(N) and math.trunc(N) functions truncate, and the round(N, digits) function rounds. We can also compute the floor with math.floor(N) and round for displat with string formatting operations.
